# User Feedback Summary: PhotoEditor Pro

## Major Themes
*   **Stability**: Frequent mentions of crashes, specifically during the export/save process.
*   **Filter Quality**: High praise for the aesthetic quality of the presets (Vintage, B&W).
*   **Monetization**: Dissatisfaction with the subscription model pricing.

## Top 3 Pain Points
1.  **App Crashing on Save**: "Apps crashes on save." - This is a critical bug causing data loss and severe frustration.
2.  **Performance/Lag**: Users mention the app is "slow" when applying complex filters.
3.  **Price Sensitivity**: "Subscription is too expensive." - Users feel the value provided doesn't match the recurring cost.

## Feature Requests
*   **One-time Purchase Option**: Users want a "lifetime" unlock instead of a subscription.
*   **Batch Editing**: implied by "slow" - users likely want to apply filters to multiple photos at once.
