# Role
You are a Frontend Developer.

# Context
We need a lightweight modal confirmation dialog with good accessibility defaults.

# Task
Provide an HTML/CSS/JavaScript snippet for a modal dialog.

# Requirements
1. **Trigger**: User clicks a "{{trigger_button_text}}" button.
2. **Interaction**: A modal appears with the message "{{modal_message}}".
3. **Controls**: Include "Confirm" and "Cancel" buttons in the modal.
4. **Accessibility**: Use `role="dialog"`, `aria-modal="true"`, and close on ESC.
5. **UX**: Clicking the backdrop closes the modal.

# Output Format
HTML/CSS/JS snippet.
